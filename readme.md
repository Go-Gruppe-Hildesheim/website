 # Go-Gruppe Hildesheim Website

Eine einfache Website für die Go-Gruppe Hildesheim, erstellt mit Astro, DaisyUI und Tailwind CSS.

## Entwicklung

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Treffen

Wir treffen uns jeden Dienstag im Deseo, einer gemütlichen Bar in Hildesheim.
